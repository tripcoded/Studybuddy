# StudyBuddy

**Team: Dark Koders**

StuddyBuddy is a personalized web application for all CS students to prepare for their career.

## Features
- Resume Generator
- Cover Letter Generator based on the job description
- Interview Preparation using Mock Interviews
- Study Material
- Roadmap to prepare for the career or develop a new skill
- Open Source Project
- Project Ideas Based on the skill level and interest
- Practice DS and Algo questions with the solution and explanation in a code editor
